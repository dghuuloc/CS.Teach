## Overview
AWS provides over 200 fully featured services organized into key infrastructure and functional categories, including Compute, Storage, Databases, Networking, and Management. Core pillars are Compute (EC2, Lambda), Storage (S3, EFS), Databases (RDS, DynamoDB), and Networking (VPC), allowing scalable, on-demand cloud solutions. 

**Core AWS Component Categories:**
* **Compute:** Services to run applications, such as Amazon EC2 (virtual servers), AWS Lambda (serverless functions), and Elastic Beanstalk.

* **Storage:** Data storage options including Amazon S3 (object storage), Amazon EBS (block storage), and Amazon S3 Glacier (archival).

* **Database:** Managed database services like Amazon RDS (relational), Amazon DynamoDB (NoSQL), and Amazon Redshift (data warehousing).

* **Networking & Content Delivery:** Tools to connect infrastructure, such as Amazon VPC (isolated networks), Elastic Load Balancing (traffic distribution), and Amazon Route 53 (DNS).

* **Security, Identity & Compliance:** Services to manage access and protect data, including AWS IAM and AWS Shield.
Management & Governance: Tools for monitoring and managing, such as AWS CloudWatch and AWS CloudTrail.
* **Analytics:** Solutions for data analysis, such as Amazon Athena and Amazon EMR.
Machine Learning/AI: Tools for AI, including Amazon SageMaker.

Other specialized categories include Internet of Things (IoT), Game Tech, Application Integration, and Media Services.
