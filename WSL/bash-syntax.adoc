= Bash Syntax

== Basic Syntax

=== Basic Script Structure

[source,bash]
----
#!/usr/bin/env bash
----

And for safer scripts, start with:

[source,bash]
----
#!/usr/bin/env bash
set -Eeuo pipefail
----

* `-e` stop on error
* `-u` fail on unset variable
* `-o pipefail` fail pipline if one command fails
* `-E` helps traps work through functions

* *Run it"*

[source,bash]
----
chmod +x hello.sh
./hello.sh
----

=== Comments

* Single-line comments:

[source,bash]
----
# This is comment
echo "Hi"
----

* There is no real multiline comment syntax. Usually people do:

[source,bash]
----
: '
This is a block of text
used like a comment
'
----
