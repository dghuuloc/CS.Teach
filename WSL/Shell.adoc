= Bash Tutorial

== Understanding Shells

A shell is a text-based interface that lets us talk to our computer.

There are different types of shells, but Bash (Bourne Again SHell) is the most popular because it's powerfull and easy to use.

*Types of Shells:*

* *Bourne Shell (sh)*: The original Unix shell, developed by Stephen Bourne.
* *C Shell (csh):* Known for its C-like syntax, popular for interactive use.
* *Korn Shell (ksh):* Combines features of sh and csh, offering advanced scripting capabilities.
* *Bash (Bourne Again SHell):* An improved version of sh, with additional features like command history and tab completion.

== Important thing

=== Overview Bash need to learn

Strong at things:

. *Using the shell interactively:* Navigate, search, inspect, chain commands, manage files, processes, permissions.
. *Writing reliable scripts:* Variables, quoting, conditionals, loops, fucntions. arguments, exit codes.
. *Using Unix tools well:* `grep`, `find`, `sed`, `awk`, `xargs`, `sort`, `cut`, `tr`, `tar`, `curl`
. * Automating real work:* Build scripts, deployment helpers, log analysis, system checks.

These are not always the same:

[source,bash]
----
#!/bin/sh
----

and

[source,bash]
----
#!/bin/bash
----

`sh` means: write portable shell script syntax.
`bash` means: we can use Bash-only features.

== Practices

* Example of a correct script

[source,bash]
----
#!/bin/bash
echo "Hello, World!"
----

Then, run:

[source,bash]
----
sh hello.sh
----

or make it executable:

[source,bash]
----
sed -i 's/\r$//' hello.sh && chmod +x hello.sh && ./hello.sh
----
